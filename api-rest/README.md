
# Zalarify API Rest

---

## Description
This is the API Rest module for the Zalarify decentralized application.


## Getting Started

```sh
# Go to the 'api-rest' folder
cd api-rest

# Install dependencies
npm install 
yarn install

# Start development live-reload server
npm run dev
yarn dev

# Start production server:
PORT=8080 npm start
```
